## Fast compiling via the terminal

### Java

Firstly, either rename source file to Main.java or change in the code `class Main` -> `class %sourceFile%`

```
javac %sourceFile%.java
java %sourceFile%
```

### C++

```
g++ -std=c++11 %source%.cpp -o %source%
./%source%
```
